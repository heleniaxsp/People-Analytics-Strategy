# Matriz Estratégica de Talento 

Este repositorio contiene una herramienta de visualización avanzada desarrollada en **R** para la segmentación estratégica de roles críticos dentro de una organización, con un enfoque en perfiles de People, Tech y Fintech.

A diferencia de las matrices de desempeño tradicionales, esta versión se enfoca en el **Diseño Organizacional y Workforce Planning** para priorizar la inversión y el desarrollo del talento.

## Parámetros de Análisis
* **Criticidad:** Impacto directo en el P&L y la ejecución estratégica del negocio.
* **Dinamismo:** Velocidad de evolución de las competencias requeridas para el rol.

## Cuadrantes de Estrategia Recomendada
1.  **Transformación:** Inversión intensiva y Upskilling.
2.  **Estabilidad (Core):** Retención y fidelización de perfiles clave.
3.  **Flexibilidad (Gig):** Modelos de contratación ágiles y proyectos por hitos.
4.  **Eficiencia (Auto):** Optimización mediante automatización con Inteligencia Artificial.

## Recursos utilizados
* **Lenguaje:** R
* **Librerías principales:** `ggplot2` y `ggrepel` (para la gestión inteligente de etiquetas visuales).

## Cómo utilizar el script
1. Clona este repositorio o descarga el archivo `.R`.
2. Ábrelo en RStudio.
3. Ejecuta el código para generar la visualización en alta resolución (`Matriz_Talento_2026.png`).
